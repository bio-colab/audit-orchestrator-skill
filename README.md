# 🎼 مِحَكّ القيادة · `audit-orchestrator`

> 🧠 بطاقة الدماغ (Obsidian): [[_brain/hubs/orchestrator|orchestrator]] · [[_brain/00-HOME|HOME]]


> قائد مجموعة مِحَكّات التدقيق الأكاديميّ — يفرز مرّةً، ويوجّه بدقّة، ويرتّب بحكمة، ويجمع بلا تكرار.

مهارةٌ لـ Claude من مجموعة **مِحَكّ (MIHAKK)** لتدقيق البحث الأكاديميّ العربيّ والعراقيّ. هذه المهارة **ليست مدقّقاً**، بل **أوركستريتر وراوتر**: تقرأ الرسالة أو البحث مرّةً واحدة، وتفرز نموذجه المعرفيّ مرّةً واحدة، ثم تشغّل **التوليفة المنطبقة** من المحكّات الثمانية **بالدرجة المناسبة** وبالترتيب الصحيح، بلا ازدواج عدٍّ عند الحدود، ثم تجمع مخرجاتها في **تقرير محكِّمٍ واحد متماسك**.

---

## لماذا قائد؟

بلا قائد، يخرج الباحث بثمانية تقاريرَ متناثرة، يعيد كلٌّ منها فرزَ العائلة تصنيفاً مختلفاً، ويكرّر العيبَ الواحد عند كلّ حدٍّ مشترك. القائد يصرف تصميماً كامناً في المجموعة أصلاً — عائلةٌ معرفية مشتركة، وبروتوكول «اعتمد تصنيف أخيك»، وحدودٌ موثّقة، و`place_marker` مُعاد، وسلّم شدّةٍ موحّد — فيحوّلها إلى خطّ إنتاجٍ واحد.

القائد يفعل شيئاً واحداً ويتقنه:

- **يفرز مرّةً** ويوزّع الناتج، فلا يُعيد أحدٌ الفرز ولا قراءة الوثيقة (يمنع انجراف إعادة الفرز).
- **يوجّه بالعائلة**، فلا يشتغل محكٌّ بلا معنى (لا «تدقيق إحصائيّ» على أطروحة فقه).
- **يرتّب بالتبعيّات** (المصادر والبنية أوّلاً، النزاهة تستهلك رايات النتائج، الإسهام يفترض حسمَ الأدبيات).
- **يمنع ازدواج العدّ** بجدول مِلكيّة السيمات.
- **يركّب** في رسالة محكِّمٍ متدرّجةٍ مع الدرجة.

القائد **لا يتجاوز** ما رخّصته كلّ مهارةٍ لنفسها: لا مصادقةَ على أصالةٍ أو أهمية، و«تعذّر التحقق» لا «غير موجود»، والنزاهة سقفها «يستوجب تحقّقاً بشرياً».

---

## المحوران: النطاق × الدرجة

المنتَج شبكةٌ ثنائيةٌ لا قائمةٌ واحدة:

- **النطاق:** نوعٌ معيّن (محكّ واحد) · شامل (التوليفة المنطبقة على العائلة).
- **الدرجة:** أساسي (الطبقة A حتمية، أوفلاين، **صفر خطر تشهير**) · وسط (A + تحقّق شبكيّ B) · متقدّم (A + B + C + تعليقات وورد + رسالة محكِّم).

الدرجة تُعرَّف بطبقات المحكّات نفسها لا بمجرّد «سريع/كامل». التفصيل في [`references/degree-translation.md`](references/degree-translation.md).

## خطّ المعالجة

```
0. الاستقبال والقراءة مرّةً واحدة      (intake-and-triage)
1. الفرز مرّةً واحدة  →  إشارةٌ حتمية ثم حسمٌ بالشجرة القانونية
2. قراءة الاختيار المنتَجيّ            (النطاق × الدرجة)
3. التوجيه / الراوتر                   (selection-matrix)
4. الترتيب بالتبعيّات                  (execution-dag)
5. التشغيل بحالةٍ مشتركة  →  ملفّ وورد واحد + ميزانيةٌ مجمَّعة
6. التركيب  →  حلّ السيمات + رسالة المحكِّم المتدرّجة   (synthesis-protocol)
```

## السكربتات

حتميّة، أوفلاين، بلا تبعيّاتٍ خارجية، ولكلٍّ اختبارٌ ذاتيّ:

```bash
# مُشير الفرز — يرفع إشارة العائلة من نصّ الوثيقة (لا حكماً)
python3 scripts/triage_scan.py رسالة.docx --json triage.json
python3 scripts/triage_scan.py --self-test

# مدقّق الملاحظات — يتحقّق من السكيما، يدمج بالموقع، يرفع تصادمات السيمات
python3 scripts/validate_findings.py ./findings --out-dir ./out
python3 scripts/validate_findings.py --self-test
```

`triage_scan.py` يقرأ `.docx` (بلا `python-docx`، عبر فكّ الأرشيف) و`.txt/.md`.
`validate_findings.py` يرفض الصياغة المصادِقة في المحاور اللاتماثلية، ويصون سقف النزاهة.

## شروطٌ للتكامل (قبل النشر الكامل)

هذه المهارة **تقود** المحكّات الثمانية، فتفترض تعديلين ضئيلين فيها:

1. **سكيما الـsidecar:** كلّ محكّ يُخرج تقريره النثريّ كالمعتاد، **ويُرفق معه** ملاحظاتٍ مهيكلة بسكيما [`references/findings-schema.md`](references/findings-schema.md). الإضافة سطرٌ واحدٌ في كلّ قالب تقرير — لأنّ المحكّات تتقاسم بنية الملاحظة نفسها أصلاً.
2. **`place_marker.py` أصلٌ مشترك:** يُنسَخ من `citation-auditor` نسخةً واحدة تعتمدها كلّ المحكّات، ولا يُعاد اختراعه (تفصيلٌ في `scripts/PLACE_MARKER_NOTE.md`).
3. **`paradigm-triage.md`:** الشجرة القانونية للفرز تُرحَّل نسخةً واحدةً إلى القائد بوصفه بيتَها الطبيعيّ (يملكها `methodology-auditor` حالياً). انسخ أصلك عند التجميع، ولا يُختلَق تفريعٌ مخالف.

## المركِّب (peer-review-synthesizer)

بروتوكول التركيب مضمَّنٌ في القائد ([`references/synthesis-protocol.md`](references/synthesis-protocol.md)) كي يكتمل خطّ المعالجة طرفاً لطرف. يمكن لاحقاً ترقيتُه مهارةً مستقلّةً تُستدعى قائمةً بذاتها على تقاريرَ جاهزة. موقفه الأخلاقيّ الأخطر — **لا شهادة سلامة**: «متقدّمٌ ونظيف» ليس «أصيلٌ ومهمٌّ ونزيه»، والمحاور اللاتماثلية لا تُصادَق أبداً.

## المبدأ الناظم
